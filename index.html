Snapshot

Mojang the makers of Minecraft release "snapshot" build during the development process. These snapshots contain the latest additions and changes to Minecraft that are not yet available in the release build. Bugs are sometimes present, the snapshots are intended for experienced users.

Vanilla

No plugins. Since the snapshots are for testing and development there is no bukkit or plugins. If you see something that looks like a plugin it is actually done with command blocks!
