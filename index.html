Mine Your Adventure

Snapshot

Mojang the makers of Minecraft release "snapshot" build during the development process. These snapshots contain the latest additions and changes to Minecraft that are not yet available in the release build. Bugs are sometimes present, the snapshots are intended for experienced users.

Installing Snapshots »

Vanilla

No plugins. Since the snapshots are for testing and development there is no bukkit or plugins. If you see something that looks like a plugin it is actually done with command blocks!

Rules

  1. Read the Rules! If you have already read the rules, read them again until you understand them.

  2. No Crying PvP is currently enabled, but that does not mean you can go around killing everyone. If you do kill someone make sure you have a reason, otherwise they may seek revenge and bring friends.

  3. No Spamming Foul language is frowned upon, but the occasional swear word is not a big deal. If you start spamming or cursing frequently you will receive a ban. Advertising other servers is strictly prohibited.

  4. No Griefing Respect others and their builds. DO NOT GRIEF.

  5. No Cheating This includes autosprint, fly hacks, x-ray texture packs…do NOT use modded/hacked clients or you will receive a ban. Client hacks/mods are strictly prohibited.

  6. Read the Rules...AGAIN!
